Lo-fi Flow 2: Гость звонит в квартиру

Спутник — домофон с дисплеем | 640x480px (горизонтальный), 2.83" TFT | Не сенсорный | v1.0 от 03.03.2026

Физические элементы панели: Камера + микрофон (сверху) · Экран 2.83" landscape (середина) · Клавиатура: цифры 0–9, кнопка вызова 🔔, кнопка отмены ✕ · Считыватель ключей RFID (большая круглая кнопка внизу) · Динамики (по бокам внизу)
Взаимодействие гостя: Набирает номер квартиры на физических кнопках 0–9 → нажимает 🔔 для вызова → говорит через микрофон/динамик. Кнопка ✕ для отмены/сброса.

1. Flow-диаграмма

Гость подходит к домофону, набирает номер квартиры на физической клавиатуре и нажимает 🔔. Абонент отвечает и решает — открыть дверь или нет.

2. Детализация экранов

Каждая карточка: wireframe экрана (landscape 640x480) + обязательные/опциональные элементы + звук + вопросы.

Обязательные элементы

  • Логотип управляющей компании
  • Время (часы : минуты)
  • Дата (день, месяц, день недели)
  • Погода (температура, состояние)

Опциональные элементы

  • Адрес, подъезд, диапазон квартир
  • Праздничный контент

Звуковая индикация

  • Нет
Доп. информация:
  • Экран всегда включён (нет ИК-датчика в v1)

Обязательные элементы

  • Набираемый номер квартиры (крупным шрифтом)
  • Подсказка: «Нажмите 🔔 для вызова»

Опциональные элементы

  • Лого УК, время
  • Подсказка «✕ — сброс»

Звуковая индикация

  • Тональный звук при нажатии каждой цифры (DTMF)
Доп. информация:
  • Ввод через физические кнопки 0–9 на панели
  • Кнопка ✕ — стереть / сбросить набор
  • Таймаут неактивности → возврат на экран ожидания
Вопросы:
  • Таймаут неактивности при наборе (сколько сек)?
  • Макс. количество цифр в номере квартиры?
  • Кнопка ✕ стирает последнюю цифру или весь номер?

3. Фоновые процессы

# Процесс Триггер Успех Ошибка / Таймаут
1 Проверка номера квартиры Гость нажал 🔔 → Вызов квартиры → Неверный номер
2 Ожидание ответа абонента Вызов отправлен → Разговор → Нет ответа
3 Решение абонента Абонент ответил → Дверь открыта → Вызов завершён
