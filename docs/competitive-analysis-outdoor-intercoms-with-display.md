## 1. 2N (Axis Communications)

### 2N IP Style
- **Экран:** 10" цветной LCD, 800x1280 (портрет)
- **Тач:** Да, емкостный
- **Защита:** IP65, IK07
- **ОС:** Проприетарная (2N OS)

#### Состояния экрана:

**Idle (ожидание):**
- Три режима idle: **Presentation** (слайдшоу до 8 изображений), **Logo** (логотип компании/ЖК), **Address** (адрес + время + погода)
- После 2 мин бездействия -- скринсейвер (яркость плавно увеличивается/уменьшается с 20-секундным циклом)
- Фоновое изображение кастомизируется (800x1280px, JPEG/PNG)

**Набор номера / Главный экран:**
- Список контактов/абонентов с полнотекстовым поиском
- Цифровая клавиатура для ввода кода доступа
- Можно настроить scrambled keypad (цифры перемешиваются после каждого ввода для безопасности)
- Иконки быстрого вызова

**Вызов:**
- Отображение статуса вызова (calling, busy)
- Пиктограмма зеленой двери -- индикатор активации замка

**Дверь открыта:**
- Зеленая пиктограмма двери на экране

**Кастомизация:**
- **2N Flexi Layout** -- встроенный визуальный конструктор UI прямо в веб-интерфейсе
- **Display API** -- сервер отправляет изображения на дисплей, получает координаты нажатий
- Темы, фоны, расположение элементов -- все настраивается

### 2N IP Verso + Touch Display Module
- **Экран:** 4" LCD, 320x214
- **Тач:** Да
- **Контраст:** 400:1, обзор 80 градусов
- **Защита:** IK07

#### Состояния экрана:

**Idle:**
- Slideshow (настраиваемые изображения/видео после таймаута)
- Логотип, реклама, объявления, тематические фото

**Главный экран:**
- Виртуальные кнопки быстрого набора
- Телефонная книга (до 10 000 контактов, древовидная структура)
- Цифровая клавиатура (обычная или scrambled)

**Вызов/Открытие:**
- Отображение статуса вызова или открытия двери

**Набор номера:**

- Отображение набираемого номера на экране

**Контакты:**
- Прокрутка списка контактов (до 2000) с помощью кнопок
- Выбор абонента для вызова

**PIN-код:**
- Ввод пароля для открытия двери

**Сообщения на экране:**
- Статус вызова
- Индикация открытия/отказа

Полноценный Android-интерфейс с:
- Списком контактов

## 3. Akuvox

### X915 (флагман)
- **Экран:** 8" IPS, 1920x1080 (портрет)
- **Тач:** Да

### X916 (топовая модель)
- **Экран:** 13" IPS, 1920x1080 (портрет)
- **Тач:** Да
- **ОС:** Android

### R29
- **Экран:** 7" тач, LCD
- **ОС:** Android
- **Защита:** IP65

### E16C
- **Экран:** 5" HD IPS, 1280x720
- **Защита:** IP65

### R28 (бюджетный с дисплеем)
- **Экран:** 3.5" TFT
- **Управление:** Кнопки

#### Состояния экранов (общие для линейки Akuvox):

**Idle / Screensaver:**
- Настраиваемый: None (экран всегда включен), Blank (темный экран), Image (загруженное изображение)
- Таймер скринсейвера: 5 сек -- 180 сек
- Пробуждение: Auto (ИК-сенсор при приближении человека) или Manual (нажатие кнопки)

**Home Screen (главный экран):**
- Темы: **Intercom**, **VMS**, **Directory**
- В теме Intercom -- 6 вкладок: **Delivery**, **Temp Key**, **PIN**, **Dial**, **Tenants/Contact**, **Reception (Speed Dial)**
- Три основные иконки (круглые, вертикально): Directory, PIN, Call
- Режимы LCD: **Default** (Call + Contacts + PIN Entry + Security Center), **Hide Contacts**, **Text Only** (пользовательский текст), **Contacts Only**
- Фон и передний план: Default, Black, White, Custom цвета

**Dial Screen (набор номера):**
- Цифровая клавиатура для набора номера квартиры
- Speed Dial -- табы/кнопки быстрого вызова без набора номера
- Отображение набираемого номера

**Contact/Directory Screen:**
- Список контактов с прокруткой
- Поиск по имени

**Calling Screen (вызов):**
- Статус вызова
- Кнопка завершения вызова (hangup)

**Talking Screen (разговор):**
- Кнопка захвата скриншота
- Клавиатура (для отправки DTMF)
- Кнопка завершения

**Door Opened (дверь открыта):**
- Текстовое уведомление: **"Open Lock Success"** (R29) / **"Unlock"** (R28)
- Голосовое сообщение: "The door is now opened"
- При открытии по RFID -- отображение информации о пользователе

**Access Denied (отказ):**
- Текстовое уведомление: **"Open Lock Failed"** (R29) / **"Unauthorized"** (R28)

**PIN Screen:**
- Цифровая клавиатура для ввода PIN-кода

- Рекламные вставки во время вызова

**Открытие двери:**
- По распознаванию лица
- По RFID-карте

## Анализ UI-паттернов по состояниям экрана

### 1. Экран ожидания (Idle)

| Паттерн | Кто использует |
|---|---|
| Слайдшоу изображений | 2N IP Style, 2N Verso, Akuvox |
| Логотип / бренд ЖК | 2N IP Style, BAS-IP, Comelit |
| Время + погода + адрес | 2N IP Style |
| Реклама / промо | BAS-IP, 2N Verso |
| Сообщения от УК | BAS-IP |
| HTML-контент / видео | BAS-IP |
| Темный экран (энергосбережение) | Akuvox (Blank mode), Hikvision |
| Пробуждение по ИК-сенсору | Akuvox |
| Номер консьержа | Fermax MILO |

**Вывод для Спутника:** Экран ожидания -- ключевая точка для рекламы, брендинга ЖК, информирования жильцов. На маленьком экране 2.83" лучше использовать минималистичный подход: время + логотип ЖК или краткое промо-сообщение.

### 2. Набор номера квартиры

| Паттерн | Кто использует |
|---|---|
| Тач-клавиатура на экране | 2N Style, Akuvox X915/X916/R29, BAS-IP AA-14FB |
| Физическая клавиатура + экран-отображение | Hikvision DS-KD-DIS, Fermax MILO/Marine, Beward |
| Тач-кнопки (не экран) + экран | Dahua VTO6531H/VTO6521H |
| Scrambled keypad (перемешанные цифры) | 2N IP Style |
| Speed Dial (быстрый вызов по одной кнопке) | Akuvox |
| Поиск контактов по имени | 2N IP Style, Akuvox, BAS-IP |

**Вывод для Спутника:** На не-сенсорном экране 2.83" набор номера будет через физическую клавиатуру (как у Hikvision DS-KD-DIS, Fermax Marine, Beward). Экран показывает набираемый номер крупным шрифтом.

## Ключевые выводы для проекта Спутник

### Размер экрана в контексте рынка
- Спутник (2.83", 480x640) будет иметь один из самых маленьких дисплеев на рынке
- Ближайшие по размеру: Hikvision DS-KD-DIS (3.5", 480x320), Fermax Marine (3.5", 320x240), Akuvox R28 (3.5")
- Важно: разрешение Спутника (480x640) ВЫШЕ чем у ближайших конкурентов, что позволяет более детальный UI

### Ориентация экрана
- Спутник: **портрет** (480x640) -- как у 2N IP Style, Akuvox X915/X916/R29/E16C
- Маленькие экраны конкурентов (3.5"-4.3") обычно в **ландшафте**
- Портретная ориентация на маленьком экране -- нестандартное решение, требует особого подхода к UI

### Что обязательно должно быть на экране:
1. **Idle:** Время + минимальная информация (логотип, адрес) или промо
2. **Набор номера:** Крупные цифры набираемого номера (ввод с физической клавиатуры)
3. **Вызов:** Номер квартиры + статус "Вызов" + таймер
4. **Разговор:** Индикатор активного разговора
5. **Открыто:** Зеленая индикация + текст подтверждения
6. **Отказ:** Красная индикация + текст ошибки
7. **RFID/Ключ:** Статус считывания + результат

### Цветовые паттерны на рынке:
- **Зеленый** = доступ разрешен / дверь открыта
- **Красный** = отказ / ошибка / блокировка
- **Темный фон** = стандарт для наружных панелей (экономия энергии, видимость на солнце)
- **Белый текст на темном фоне** = максимальная читаемость
- **Акцентные цвета** (зеленый, синий) = активные элементы

### Уникальные возможности конкурентов для изучения:
- **BAS-IP:** реклама, видео, HTML-контент, аватары жильцов -- максимально богатый контент на экране
- **2N IP Style:** полная кастомизация через Flexi Layout и Display API -- гибкость
- **Akuvox:** продуманная система состояний с текстовыми промптами при открытии/отказе
- **Comelit:** элегантная темная тема с минимум элементов -- хороший пример для маленького экрана
